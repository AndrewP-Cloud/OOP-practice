PROJECT TITLE: OOP_principlesPractice
PURPOSE OF PROJECT: Allow me to review the OOP principles in a functional and simple program.
VERSION or DATE: 1.1
AUTHORS: Andrew Perry
USER INSTRUCTIONS: Use the Test file to alter the parameters to create and use different 
accounts. Error handling, database storage, algorithms and other more advanced concepts will be left
out so as not to waste time on this simple program meant for review.

This project was originally completed for a class in my final semester of college. It was mythe 
second OOP Java programming class, but it was much simpler and focused on the OOP principles
more than the computer science that was focused on in my programming for engineers class.

The intentions of this were to demonstrate that I understand the basic principles of OOP programming
and to help me warm up before tackling my next project. An application built on the .NET stack.
